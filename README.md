#stock_MultiLayerNet  
何日間か過去の株価と単純移動平均との差、月、業種  
から次の日の株価が上がるか下がるかを予測する

#requierments 
numpy  
pandas  

#How to use  
`python get_make_data/get_code_num.py  銘柄番号の取得`  
`python get_make_data/get_stock_data.py  株価データの取得`  
`python get_make_data/Granville_low.py  データの整形`  
`python main.py  学習`
